# Ejecutar en desarrollo

1. Clonar el respositorio
```
git clone https://github.com/ 
```

2. Descargar paquetes
```
yarn 
```

3. Clonar archivo ```.en.template``` y renombrarlo a ```.env``` 
<br/>

4. Levantar la base de datos
```
docker-compose up -d
```



## Stack usado
* MongoDB
* Express JS
* TypeScript
